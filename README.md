# 🏆 Clash Commander Bot

<img src="assets/clan-logo.png" width="400" alt="Clan Logo"/>

A modular Discord bot built for elite Clash of Clans clans.

> Automatically assign Town Hall roles, track players, manage bases, and prep for war — all in one clean, expandable bot.

---

## 💡 Features

- 🔗 `/link @user #tag` — Link Discord users to Clash accounts via API
- 🏰 Auto-assign TH roles (TH13–TH17 ready)
- 🧱 `/addbase` — Organize base links by TH and layout style (ring, box, diamond, etc.)
- 📂 JSON-backed data storage with clean role tagging
- ⚔️ Ready for expansion: war tracking, raid manager, and more
